# World Pulse

国際情勢ニュースをポップにまとめるアプリ。ニュース取得は無料、AIによる背景解説はAPIキーを設定するまでオフ（課金なし）です。

## 必要なもの

- [Node.js](https://nodejs.org/)（18以上）※ 無料
- （あとで解説機能を使う場合のみ）Anthropic Consoleで発行するAPIキー

## セットアップ（無料でここまで動きます）

ターミナルでこのフォルダに移動して、以下を順番に実行してください。

```bash
npm install
npm start
```

起動したら、ブラウザで以下を開いてください。

```
http://localhost:3000
```

ニュース一覧がリアルタイムで表示されます。この状態では課金は一切発生しません。

## 背景解説を有効にする（ここから課金対象）

1. [Anthropic Console](https://console.anthropic.com/) でアカウントを作成し、APIキーを発行する（少額をチャージするだけで使えます）
2. `.env.example` を `.env` という名前でコピーする
3. `.env` の中の `ANTHROPIC_API_KEY=` の後ろに、発行したキーを貼り付ける
4. サーバーを再起動する（`npm start`）

これで「💡 前提を解説」ボタンを押すと、AIが背景解説を生成するようになります。課金されるのは、このボタンを押して解説を生成したときだけです。

## 気に入らなかったら

`.env` を削除するか、`ANTHROPIC_API_KEY` を空にしておけば、いつでも無料の状態（ニュース一覧のみ）に戻せます。
